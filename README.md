# Healthcare Management System

A comprehensive healthcare management platform built with ASP.NET Core Web API, SQL Server, and React frontend.

## Project Structure

### Backend
- ASP.NET Core 7.0 Web API
- SQL Server Database
- Entity Framework Core ORM
- JWT Authentication
- Clean Architecture Pattern

### Frontend
- React with TypeScript
- Axios for API calls
- React Router for navigation
- Responsive UI with modern design

## Features

### Core Modules
1. **Patient Management** - Register, manage patient records
2. **Doctor Management** - Doctor profiles and specializations
3. **Appointment Scheduling** - Book, reschedule, cancel appointments
4. **Medical Records** - Store and retrieve patient medical history
5. **Prescription Management** - Manage medications and prescriptions
6. **Billing & Payments** - Invoice generation and payment tracking
7. **Staff Management** - HR and staff administration
8. **User Authentication** - Secure login with JWT tokens

## Getting Started

### Prerequisites
- .NET 7.0 SDK
- SQL Server 2019 or higher
- Node.js 16+
- npm or yarn

### Installation

See individual README files in Backend and Frontend directories.
